What is CSS and JS?

Css

Cascading Style Sheets or CSS is a way to style HTML in a convenient way while being able to maintain a simple page structure. As seen from Design 1 and Design 2 on the page, a website with exactly the same layout can be shown in a vastly different way with different features and animations possible. This can be handled via a single file for a whole web stucture using class and id selectors as well as selecting whole tags and sub structures within a website.

JS

Javascript or ECMAscript is a scripting language that is mainly used on the backend of websites to take in and manipulate data that could be shown on the screen. In the case of the coffee applications, javascript can be used to get the data from forms on the web page, apply processing onto that data and output back onto the website for a fluent and easy to manage system.

Designs

Design 1

This design incorperates a side navigation bar on the left that is fixed in position and can be expanded when hovered. This allows the easiest browsing of content of the page while also giving easy access to the set of navigation links. The main page follows a grid flexbox design showing how text can wrap in rows together and span across multiple columns dynamically. The header animation easing in with the font size displays the title in a dynamic way and is instantly visible on the page load.

Design 2

Design 2 instead opts for a top sticky navigation bar showing text only until hovered to display the relevant svg icons. The block design of this page allows a much easier linear experience to follow while the images disable wrap to give a form of structure in each block. The backgrounds to main block images expand when hovered. The main background colour of the page utilises a gradient rather than a fixed colour with the background colour of blocks also having lower opacity to reflect this. The movement of the central image shows how images can be translated across the page and moved dynamically.
